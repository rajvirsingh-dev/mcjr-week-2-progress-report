3.3 Data Collection Methodology

Phase 1 – Statistical Parameter Extraction
Division-level NAT Mean Percentage Score (MPS) statistics, resource ratios, and annual enrollment figures were manually extracted from DepEd published reports using PDF parsing with pdfplumber.
Phase 2 – Synthetic Record Generation
Since DepEd BEIS student-level microdata is restricted to authorized LGU and research users, individual records were synthesized using the extracted division parameters (means/std from actual distributions). This approach is standard in Philippine educational research (Orbeta et al., 2023, PIDS Discussion Paper 2023-01; Mapa et al., 2022, PSA Statistical Research).
Phase 3 – Intentional Data-Quality Injection
To simulate real-world data-entry issues, ~4% missing values, ~1% duplicates, and ~0.5% out-of-range score errors were injected before the preprocessing step to validate the cleaning pipeline.

5. Data Preprocessing Pipeline

5.1 Preprocessing Steps

Step 1 – Duplicate Removal
Used pandas.drop_duplicates() to identify and remove all ~5,300 injected duplicate records, reducing the dataset to the clean baseline.
Step 2 – Out-of-Range Score Correction
All subject score columns were clipped to the valid range [0, 100] using pandas.clip(). ~2,650 out-of-range values in math_score (>100) and science_score (<0) were corrected.
Step 3 – Missing Value Imputation
Missing numerical values in math_score, science_score, english_score, teacher_count, and textbook_adequacy_% were filled using the group median, grouped by (school_year, division) to respect temporal and geographic context. A global median fallback handled any all-NaN groups.
Step 4 – NAT MPS Recomputation
The nat_mps field was recomputed from cleaned subject scores: average of Math, Science, English, Filipino, AP, and MAPEH.
Step 5 – Data Type Optimization
Categorical columns converted to pandas Categorical dtype. Integer columns cast to int8 / Int16, reducing memory usage by ~28%.
Step 6 – Performance Band Feature Engineering
New feature performance_band derived from nat_mps using DepEd MPS performance levels:
  • Low (< 50)
  • Approaching Proficiency (50–60)
  • Proficient (60–75)
  • Advanced (> 75)
Step 7 – Resource Index Feature Engineering
Composite resource_index (0–100 scale):
  • Student-Teacher Ratio component — 40% weight
  • Textbook Adequacy component — 40% weight
  • Internet Connectivity component — 20% weight
Step 8 – School Year Ordinal Encoding
sy_ordinal column (0–4) added for use in trend regression modeling in Week 3.

6. Exploratory Data Analysis (EDA)

6.1 EDA Before Preprocessing – Key Findings

  • Missing Value Map: Heatmap of 3,000 sampled rows revealed scattered missingness (~3–4%) in math_score, science_score, teacher_count, and textbook_adequacy_%.
  • Out-of-Range Values: Box plots clearly showed values exceeding 100 in math_score and negative values in science_score.
  • Duplicate Detection: 1.0% of records were flagged as full duplicates.
  • Province Distribution: Camarines Sur had the highest record count (~200,000+), followed by Albay (~140,000+), reflecting the DepEd school directory distribution.
